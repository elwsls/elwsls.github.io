Mac 添加打印机部门管理 ID

2020-09-11 1 min read

背景

macOS Catalina 10.15.6
Kyocera ECOSYS M4125idn

现象

打印机设置了部门管理
mac 隔空打印错误

分析

mac 未添加部门管理ID

方案

驱动
官网下载安装

通过 IP 地址重新添加打印机
IP 地址在 IPv4 设定中查询 =》选择行式打印机监控程序 - LPD =》选择 Optional disk: Not installed

从打印页面进入驱动设置
打印 =》显示详细信息 =》Print panel =》Settings

添加部门管理ID
Job =》Job accounting: =》Account ID: =》填写部门管理 ID =》OK
